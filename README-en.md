# Bujidao Web App

[简体中文](./README.md) | [English](./README-en.md)

[Official Website](https://bujidao.org/) [Telegram](https://t.me/bujidaoChat)

A modern web modular theme system built with Vue 3, featuring responsive design for multiple devices.

## Technology Stack

- **Framework**: Vue 3.5+ (Composition API)
- **Language**: TypeScript
- **Build Tool**: Vite (rolldown-vite)
- **State Management**: Pinia + pinia-plugin-persistedstate
- **Routing**: Vue Router 4
- **UI Component Library**: Element Plus
- **Charts**: ECharts
- **Styling**: Sass/SCSS
- **Code Style**: Prettier
